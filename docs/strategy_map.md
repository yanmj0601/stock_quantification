# 量化策略地图

这份地图分两层：

- 已接入工程并可直接回测的策略
- 主流但当前工程尚未正式实现的策略

## 1. 已接入工程并可回测

### A 股

- `A股基线增强`
  - 家族：指数增强
  - 特征：中期相对强弱 + 盈利能力 + 风险控制
- `A股动量核心`
  - 家族：动量
  - 特征：20/60 日相对强度 + 趋势
- `A股质量动量`
  - 家族：质量 + 动量
  - 特征：盈利能力叠加中期动量
- `A股低波防御`
  - 家族：低波动
  - 特征：低波动、低回撤、少量趋势
- `A股流动性龙头`
  - 家族：流动性
  - 特征：高成交活跃度 + 中期强势

### 美股

- `美股基线质量动量`
  - 家族：质量 + 动量
  - 特征：相对强度 + 盈利能力 + 经营质量
- `美股动量核心`
  - 家族：动量
  - 特征：20/60 日相对强弱 + 趋势 + 流动性
- `美股质量精选`
  - 家族：质量
  - 特征：盈利能力 + 经营质量 + 回撤控制
- `美股低波防御`
  - 家族：低波动
  - 特征：质量、盈利能力和低波动约束
- `美股流动性龙头`
  - 家族：流动性
  - 特征：高流动性大市值股票 + 中期强势

## 2. 主流但当前仍属于路线图

这些方向是主流量化框架里常见的，但当前项目还没有完整实现：

- `价值因子`
  - 需要更稳定的估值与财报数据口径，例如 `PE / PB / EV/EBITDA / FCF`
- `均值回归`
  - 需要更短周期收益、乖离率、分位数和更细的交易成本模型
- `事件驱动`
  - 需要公告、财报、回购、指数组样变更等结构化事件数据
- `市场中性 / 多空对冲`
  - 需要空头、融券、beta/行业中性和更严格的组合优化
- `统计套利`
  - 需要配对池、协整检验、价差建模和更高频的成交建模
- `CTA / 多资产趋势`
  - 需要期货、外汇、商品等多资产数据和保证金语义
- `机器学习选股`
  - 需要更长历史、更稳定特征库和更严格的样本外验证

## 3. 难度与适合度

### 最适合当前项目优先推进

- 中低频多因子 `long-only`
- 指数增强
- 低波动 / 质量增强
- 流动性筛选 + 动量

### 适合后续研究但不建议当前优先

- 事件驱动
- 均值回归
- 机器学习排序

### 暂不建议当前阶段优先

- 市场中性
- 统计套利
- CTA

## 4. 当前工程建议顺序

1. 先在已实现的 `long-only` 策略里做稳定性比较
2. 继续做因子消融和更长区间样本外验证
3. 再逐步引入价值、事件、机器学习等更复杂方向
